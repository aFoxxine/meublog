O que é algoritmo?

Manzano (ss2009) define: “Algoritmos são conjuntos de passos finitos e organizados que, quando executados, resolvem um determinado problema”. O algoritmo é utilizado em praticamente todas as áreas do conhecimento, mesmo sem utilizar o nome algoritmo. Alguns exemplos de algoritmos são: fazer uma busca no Google, fazer compras online, jogar, usar o celular (para conversar com pessoas, compartilhar experiências, GPS). Algoritmo também pode ser resumido em rotina, no dicionário: sequência dos procedimentos, dos costumes habituais, na informática: reunião dos direcionamentos que auxiliam na execução de uma tarefa.

Algoritmo computacional é um conjunto de passos a serem seguidos por um módulo processador e seus respectivos usuários que, quando executados na ordem correta, conseguem realizar determinada tarefa. Existem diversas formas de representar um algoritmo. As três formas mais utilizadas são: descrição narrativa, fluxograma e pseudocódigo. Assim que o projeto de um programa de computador está concluído, é necessário transformá-lo em um software. Para tanto, faz-se a tradução do projeto definido para uma linguagem de programação formal, aquela que é executada em um computador.

A linguagem C

A linguagem C foi criada em 1972 por Dennis Ritchie no Bell Telephone Laboratories. Influenciou diversas linguagens como Java, PHP, Pascal, entre outras. A linguagem C não possui comandos de escrita e leitura, para resolver o problema existe um conjunto de funções na Biblioteca de Funções, como por exemplo: stdio.h (standard input/output) stdlib.h (standard library). A linguagem C é Case Sensitive, isto é, faz diferenciação entre maiúsculas e minúsculas, por exemplo: não é a mesma coisa escrever Main(), main(), MAIN(), mAIN(). Agora, vamos apresentar alguns elementos e comandos da linguagem C.

Chaves ({ }):

São usadas para marcar início e fim de programas, estruturas condicionais, estruturas de repetição, classes, métodos e outros elementos que fazem parte do código.

Ponto e vírgula (;):

Esse sinal indica o fim da instrução e deve ser colocado em todo final de linha que envolva algum processamento direto de dados tais como atribuição, operações matemática, etc.

Variáveis:

É necessário declarar as variáveis que serão utilizadas durante a execução do programa. O nome de uma variável pode ser constituído por letras do alfabeto (minúsculas ou maiúsculas), dígitos (0 ... 9) e ainda pelo caractere underscore (_), o primeiro caractere terá que ser uma letra, não pode ter por nome uma palavra reservada da própria linguagem C, por exemplo: printf. A leitura e escrita de variáveis podem ser realizadas através das funções scanf (leitura) e printf (escrita), utilizando os formatos adequados para serem lidos ou escritos. Os seus formatos são: int (inteiro), float, double, char (caractere) e bool.

Operadores:

Um operador é um símbolo usado para criar expressões, a partir de constantes e variáveis. Os operadores da Linguagem C e suas funções são: + para soma, - para subtração, * para multiplicação, / para divisão, % para resto da divisão, POW() para exponenciação e SQRT() para raiz quadrada. Existem também os operadores relacionais, que são: = para igualdade, != diferença, < para menor, > para maior, <= para menor ou igual e >= para maior ou igual. Além desses, temos os operadores lógicos: && para e, || para ou e ! para negação.

scanf:

É função para entrada de dados na Linguagem C.

printf:

É a função para saída de dados na Linguagem C.

if:

É uma das instruções de controle de fluxo. A instrução if funciona assim: A condição é avaliada, se o resultado da condição é verdadeira, executa a instrução condicionada, senão, executa a próxima instrução ou termina.

else:

É uma das instruçõesde controle de fluxo, também. É uma instrução que anda junto com a if, ela executa a instrução, caso a do if seja falsa.

else if:

Existem situações em que o teste de uma condição não é suficiente para tomar uma decisão. Pode ser necessário testar mais do que uma condição, então usa-se o else if para indicar uma nova instrução condicionada.

while:

É uma estrutura de repetição ou laço de repetição, uma estrutura de programação que facilita repetir determinados trechos de código, reduzindo o trabalho de programação, principalmente quando é preciso repetir várias vezes alguma ação importante no programa. A estrutura de repetição faz a verificação da condição para executar as instruções subordinadas de um bloco adjacente após conferir a validade do resultado lógico de uma condição estabelecida. Sendo o resultado lógico da condição verdadeiro, são executadas as instruções subordinadas do bloco adjacente. No momento em que o resultado lógico da condição não é mais verdadeiro, o laço de repetição é automaticamente encerrado.

for:

É outra estrutura de repetição da Linguagem C.
